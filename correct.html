About Me

Front-End Developer

I'm Masroor Ibrahim, a front-end developer specialising in crafting responsive and accessible websites using the core building blocks of the web: HTML and CSS. I turn design features into reality, ensuring every site i build is not only visually stunning but also inclusive and perfomant. What excited me most about Front-End Development is the power of CSS. With it, I can create intricate layouts, smooth animations and responsive designs that adapt seamlessly across device. I believe in wring clean, semantic HTML to ensure accessibility and a solid foundation for all users, including those using assistive technologies. My expertise lies in leveraging modern CSS techniques like Grid and Flexbox to build complex layouts, while maintaining a mobile first approach. i take pride in writing efficient, maintainable without relying on javascript for styling, resulting in faster load times and a more reliable user expereince. Whem am not coding you can find me exploring the latest design trends, hiking in the mountains or contributing to open source projects. Am always eager to take on new challenges and collaborate on projects that make difference.
